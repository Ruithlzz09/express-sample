## Tech Stack:
1. nodeJS (>=20.10.0)
2. express
3. npm

Note: Please use npm only.

## How to start:
1. check node version compatiblity from package.json
2. create .env file , by reference of .env.sample and setup all required key values
3. Run: npm run build
4. Run: npm run dev
